# door_lock_controller_esp32

这是一个非常简单的嵌入式试水作，用于解锁和上锁宿舍的门锁旋钮。

我们宿舍的门锁是一个非常简单的那种房间门，在已经解锁了的情况下，只需要拉动门把手的那种。该项目做的东西其实非常简单，只用控制门锁旋钮的开关，围绕着这个旋钮做开发。

## 已实现的功能

- [ ] 通过蓝牙开关锁
- [ ] 通过NFC开关锁

## 使用的材料

- ESP32C3

- MG995/MG996舵机（360度）

- 齿轮模型

- 面包板，连接线

## 使用步骤

1. 接好各种零件到面包板，舵机接GPIO??，固定好舵机和门锁旋钮外层模型
2. 通电，刷入固件
3. 使用蓝牙连接设备（建议用“蓝牙串口SPP”），输入终端出现的配对码配对
